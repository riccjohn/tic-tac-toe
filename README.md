## Installation

- Clone the repo, install dependencies
  - `git clone git@github.com:riccjohn/tic-tac-toe.git`
  - `cd tic-tac-toe`
  - `npm i`


- Testing
`npm test` to run tests once
`npm run test-watch` to watch all tests for changes

- Start the app
  - `npm start`
  - Open browser and go to `localhost:3000`
